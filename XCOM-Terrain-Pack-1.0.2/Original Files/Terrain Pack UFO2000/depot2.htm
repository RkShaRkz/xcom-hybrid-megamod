The UFO2000 Map Depot

UFO2000 can support additional maps created by users. In order to use these maps you are required to have beta version 0.7.1031 or newer.
Some maps only require the UFO Demo to be used in UFO2000, some full UFO Defense installation, some full UFO Defense and TFTD installation. Installation of the maps is simple. - The files only have to be unzipped into the /extensions directory in your UFO2000 folder. See the README files for specific instructions.


Click on the pictures for a larger view, and on the title to download.
